Консольное приложение для учёта животных в Московском зоопарке. 

Поддерживает:
* добавление новых животных с проверкой здоровья через ветеринарную клинику
* расчёт общего потребления еды
* формирование списка животных для контактного зоопарка (травоядные с добротой > 5)
* вывод инвентарных номеров всех учтённых сущностей


Применение SOLID:

S (Single Responsibility Principle)
Каждый класс отвечает за одну задачу:
* Zoo — управление коллекцией,
* VeterinaryClinic — проверка здоровья,
* Menu — взаимодействие с пользователем,
* AnimalFactory — создание животных.

O (Open/Closed Principle)
Система открыта для расширения: для добавления нового животного достаточно создать класс, унаследованный от Animal, и реализовать необходимые интерфейсы.

L (Liskov Substitution Principle)
Любой подкласс Animal может быть использован вместо базового типа без нарушения логики (например, при расчёте еды или проверке здоровья).

I (Interface Segregation Principle)
Используются узкоспециализированные интерфейсы:
* IAlive — для живых сущностей (свойство Food),
* IInventory — для инвентаризуемых объектов (свойство Number),
* IContactable — для определения возможности взаимодействия с посетителями (свойство Kindness).
* IZoo - для описания функций зоопарка
* IVetClinic - для описания функций вет. клиники
* IAliveFabric - для описания функционала фабрики создания животных
----------------------------------------
* IMenu - для описания основных способов взаимодействия с пользователем
* IMessaging - для описания основных типов сообщения информации пользователю

D (Dependency Inversion Principle)
Зависимости (IVetClinic, IAnimalFactory) внедряются через конструктор.
Классы зависят от абстракций, а не от конкретных реализаций.


Внедрение зависимостей (DI)
Для управления зависимостями используется встроенный DI-контейнер Microsoft.Extensions.DependencyInjection.
Все сервисы (Zoo, AnimalFactory, Menu, и т.д.) регистрируются в контейнере при старте приложения.
Это упрощает тестирование и соблюдение принципа инверсии зависимостей.


Инструкция по запуску:
1. Убедитесь, что установлен .NET 8
2. Запустите консольное приложение ZooSD
3. Следуйте инструкциям в консоли
